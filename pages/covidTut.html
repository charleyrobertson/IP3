Covid-19 Statistics Tutorial



On this page, there will be a tutorial on how to utilise the Covid-19 M-M-Media API. This API provides many different details about Covid-19. This tutorial will focus specifically on the amount of cases, recovery cases and deaths per country that the API provides statistics for. It will contain how to code the API and how to use it the way that I have decided to implement it.
JavaScript will be used to pull the data from the API's and a HTML page will be used to display the data that is pulled from the API. So it was important to create a two JavaScript files and a HTML file within the program. Within the HTML page, you need to call JavaScript pages so that it will run correctly. I called my JavaScript pages 'covidDS.js' and 'countryDS.js', so this is the page that I had to call. You also have to call the link to AJAX as this will be used to utilise the API's. Refer to the image on the left hand side to see how this is done.
As mentioned in the above section of the tutorial. The AJAX link is used in the HTML document so that we can use it. This allows us to tell the document in the JavaScript file to run automatically as soon as it is called. To be able to utilise the Covid-19 API, the REST countries API needs to have the country data pulled out first.
