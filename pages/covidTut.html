To get the corona virus API to work it was important to source an API that contained every country within it and compare the countries and ensure that each country that the API will provide corona virus data for was included within this API.

I used AJAX to pull information from the API and display each country within a drop down menu. When the drop down menu is selected, the country will be pulled in by the program and inserted into the corona virus API to retrieve the cases, recovery rate, deaths and the time that it was last updated.
