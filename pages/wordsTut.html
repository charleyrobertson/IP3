Words & Synonyms Tutorial

Why Words & Synonyms?

While the visualisation of a Words/Synonyms API was not my initial intention, the sheer quantity of words out there allows for a massive range of visualisation. The data made available from the many different types of Words API queries could be manipulated many ways, and the simplest and most interesting way happened to be a definition/synonym query, which would be visualised in a Polar Area chart.

The API/JS in use:

The Words API was chosen as it grants access to a large set of data, one that contains over 150,000 words that each have multiple values/variables associated them that can be heavily manipulated for a wide range of utilisations.
The Chart.js Polar Area Chart was used to allow for the visualisation of the dataset given through the words API. Chart.js comes with extremely flexible JavaScript charting that allows for very different types of datasets to be visualised however they need be. A Polar Area Chart suited our specific query the best.

Displaying the Data (Chart JS):

- Sets all the data in the chart to the 'synLengthArray', which is an array of the amount of synonyms associated with the word.
- Sets all the labels in the chart to the 'defArray', which is an array of all the words definitions.
- Sets the type of chart to a polar area chart, with the legend & scale included.
