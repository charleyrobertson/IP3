Similarily to the REST Country API. The data has to be pulled from the M-M-Media Covid-19 API also. This will be done using the same getJSON() method that was used previously. However, since there is more than one datafield within each section of this API, it is important to pull out the specific data that needs to be used within the page and the chart. In this instance, the data that is needed is the confirmed cases, recovery cases, deaths and the time the API was last updated. To do this, there needs to be a country to then pull the data for that specific country; the search for the country will be pulled from the drop down menu. Refer to image on how to do this.
Now that we have the data needed to populate the text fields. The data has to be passed through into a method that will use DOM manipulation to populate the text fields. The method will be called using 'displayData(data)' whilst passing through the data.
